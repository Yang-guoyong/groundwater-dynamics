# groundwater-dynamics (地下水动力学）

讲了多年的《地下水动力学》课程，课纲随学时设置频繁变化，学生编程语言也从较早的 Fortran 变成 VisualBasic 及 VBA, 最新的要求是掌握 Python, 将个人对课程的理解形成讲稿，程序也更新为 Python。

程序环境可以根据《地下水动力学》实验指导书 配置.

教材参考 薛禹群.地下水动力学[M].北京：地质出版社.
